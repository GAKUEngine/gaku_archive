gaku_archive
=========

Archive functionality for GAKUEngine


### Install

Add to your `Gemfile` :

```ruby
gem 'gaku', github: 'GAKUEngine/gaku'
gem 'gaku_archive', github: 'GAKUEngine/gaku_archive'
```

Install dependencies: 

    $ bundle install

Run `gaku_archive` generator :

    $ rails g gaku_archive:install 

Run the server: 

    $ rails s
